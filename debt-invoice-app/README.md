# تطبيق إدارة الفواتير والديون

تطبيق موبايل متكامل لإدارة الديون والفواتير للمشتركين، مبني باستخدام **EXPO** و **React Native** مع ربط مع **SAS Radius Server**.

## المميزات الجديدة 🎉

### 🗄️ قاعدة بيانات SQLite محلية
- تخزين معلومات السيرفرات المتعددة
- تخزين مؤقت للمشتركين والأجهزة
- تخزين الفواتير للوصول السريع

### 🖥️ إدارة السيرفرات
- إضافة سيرفرات SAS متعددة
- تعيين سيرفر افتراضي
- اختبار الاتصال قبل الحفظ
- مزامنة البيانات من السيرفر

### 👥 إدارة المشتركين
- قائمة المشتركين الكاملة
- البحث والتصفية حسب الحالة
- عرض تفاصيل المشترك
- عمليات الإيداع والسحب

### 📄 إدارة الفواتير
- قائمة الفواتير المحلية
- تصفية حسب حالة الدفع
- البحث في الفواتير

## التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|----------|
| EXPO | إطار عمل React Native |

## تثبيت التطبيق

### المتطلبات
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn أو bun
- EXPO CLI

### خطوات التثبيت

```bash
# الدخول لمجلد المشروع
cd debt-invoice-app

# تثبيت الحزم
npm install

# تشغيل التطبيق
npx expo start
```

### تشغيل على أجهزة مختلفة

```bash
# تشغيل على Android
npx expo start --android

# تشغيل على iOS
npx expo start --ios

# تشغيل على الويب
npx expo start --web
```

## استخدام التطبيق

### 1. إضافة سيرفر
1. افتح التطبيق واذهب إلى تبويب "السيرفرات"
2. اضغط على زر + لإضافة سيرفر جديد
3. أدخل المعلومات:
   - **اسم السيرفر**: اسم تعريفي (مثال: السيرفر الرئيسي)
   - **عنوان URL**: رابط السيرفر (مثال: http://demo.sasradius.com)
   - **اسم المستخدم**: اسم مدير النظام
   - **كلمة المرور**: كلمة مرور المدير
4. اضغط "حفظ"

### 2. الاتصال بالسيرفر
1. من قائمة السيرفرات، اضغط على السيرفر المطلوب
2. سيتم محاولة الاتصال تلقائياً
3. عند نجاح الاتصال ستظهر علامة خضراء

### 3. مزامنة البيانات
1. بعد الاتصال بالسيرفر
2. اضغط على زر المزامنة 🔄
3. سيتم تحميل المشتركين والفواتير

### 4. إدارة المشتركين
1. اذهب إلى تبويب "المشتركين"
2. استخدم البحث للعثور على مشترك
3. اضغط على المشترك لعرض التفاصيل
4. يمكنك إيداع أو سحب رصيد

## واجهة API المستخدمة

### Admin API
| Endpoint | الوصف |
|----------|-------|
| `POST /login` | تسجيل الدخول |
| `GET /index/user` | قائمة المشتركين |
| `GET /index/online` | المستخدمين المتصلين |
| `GET /index/invoice` | قائمة الفواتير |
| `POST /user/deposit` | إيداع رصيد |
| `POST /user/withdraw` | سحب رصيد |

## قاعدة البيانات

### الجداول
- **servers**: تخزين معلومات السيرفرات
- **cached_users**: تخزين المشتركين محلياً
- **cached_devices**: تخزين الأجهزة
- **cached_invoices**: تخزين الفواتير
